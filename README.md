## 🚀 Projetos em Destaque

### 📺 [Projeto Mídia Indoor](https://github.com/MiguelEstevam/projeto-midia-indoor)
Sistema de mídia indoor com **React** no frontend e **Supabase** como backend. Focado na gestão e exibição dinâmica de conteúdos em telas digitais.

### 🎲 [Jogo de Tabuleiro](https://github.com/MiguelEstevam/Jogo-De-Tabuleiro)
Jogo desenvolvido em **Python** com aplicação de estruturas condicionais, loops e lógica de programação — projeto acadêmico com foco em fundamentos.

### 🍽️ [Gerenciador de Restaurante](https://github.com/MiguelEstevam/Projeto-1-de-Aed1-UCL)
Primeiro projeto em **C#** na UCL: sistema de gerenciamento para restaurante mineiro com controle de mesas, cardápio e orientação a objetos.

Projeto com pipeline de **CI/CD** em Python para comparação e avaliação de dados de clientes, aplicando boas práticas de DevOps.
